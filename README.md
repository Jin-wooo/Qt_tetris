# TETRIS With PyQt5

## Introduction
> Sample Tetris game made with PyQt. Got many hints from http://zetcode.com/gui/pyqt5/tetris/
 
## How to play
this game use tenkey.
+ 4 - Move left
+ 6 - Move right
+ 7 - Rotate left
+ 9 - Rotate right
+ 8 - Falldown immediately

## Development Environmnet
+ PyQt5
+ py2exe
+ PyCharm Community Edition 2020.1.2 x64
